Basic functions

Console functions (print, warn, error)

-- 1 line comment
--[[ Multi-line comments ]]-- - Any comments will start with -- and will be ignored by decompiler
print("Hello, world!") - outputs a message to the console.
warn("Hello, world!") - outputs a warning message to the console.
error("Hello, world!") - raises an error and stops the script execution.
You can change "Hello, world!" to any text you want or even numbers, like:
print(123)

Variable declaration (local, global)

local x = 10 - declares a local variable x with the value of 10.
y = 20 - declares a global variable y with the value of 20.
Local variables are only accessible within the block of code they are declared in, while global variables can be accessed from anywhere in the script.
Usualy using for print or something like:
local name = "Jack"
print("Hello, " .. name) - outputs "Hello, Jack" to the console.

Function declaration (function)

function YourFunctonName(Args)
print(Args)
end
YourFunctonName("Hello, world!") - declares a function named YourFunctionName that takes one argument Args and prints it to the console when called.
